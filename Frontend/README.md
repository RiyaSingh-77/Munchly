# Munchly 🍔

A full-stack food delivery web app where users can discover food, and food partners can list their meals.

## Live Demo
[munchly-dev-feed.vercel.app](https://munchly-dev-feed.vercel.app)

##  Tech Stack
- **Frontend:** React, Vite, Axios, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Media Storage:** ImageKit
- **Deployment:** Vercel (Frontend), Render (Backend)

##  Features
- User & Food Partner authentication
- Upload food videos/images
- Like and save food items
- Food partner profile & dashboard
- Responsive UI
